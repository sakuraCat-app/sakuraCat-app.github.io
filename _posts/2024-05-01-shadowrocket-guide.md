## 什么是小火箭？

小火箭（Shadowrocket）是 iOS 平台上最受欢迎的代理工具，支持多种协议和规则配置。它提供了强大的功能和友好的用户界面。

## 安装小火箭

### 方式一：App Store 购买
1. 打开 App Store
2. 搜索 "Shadowrocket"
3. 点击购买（约 $2.99）
4. 使用 Apple ID 完成支付
5. 等待安装完成

### 方式二：使用礼品卡
1. 购买 App Store 礼品卡
2. 兑换到您的账户
3. 使用余额购买应用

## 基本配置

### 添加节点

#### 方式一：手动添加
1. 打开小火箭
2. 点击"+"按钮
3. 选择节点类型（SS、Trojan、VMess 等）
4. 填写服务器信息
5. 点击"完成"

#### 方式二：扫描二维码
1. 打开小火箭
2. 点击"+"按钮
3. 选择"扫描二维码"
4. 扫描节点二维码
5. 点击"完成"

#### 方式三：导入订阅
1. 复制订阅链接
2. 打开小火箭
3. 点击"配置"选项卡
4. 点击"+"按钮
5. 选择"从 URL 导入"
6. 粘贴订阅链接
7. 点击"完成"

### 连接节点

```
首页 → 选择节点 → 点击"连接"按钮 → 允许 VPN 配置
```

## 高级功能

### 1. 规则配置

小火箭内置多种规则集：
- **直连** - 国内流量不走代理
- **代理** - 所有流量都走代理
- **智能路由** - 根据规则自动分流

### 2. 分流规则

```
配置 → 规则 → 选择或自定义规则
```

常用规则：
- 国内网站直连
- 国外网站代理
- 特定应用代理

### 3. 延迟测试

```
首页 → 长按节点 → 选择"测试延迟"
```

选择延迟最低的节点以获得最佳速度。

## 常见问题

### Q: 小火箭无法连接？
A: 
1. 检查节点是否过期
2. 尝试更换节点
3. 重启应用
4. 检查网络连接

### Q: 如何设置自动选择？
A:
1. 点击"配置"选项卡
2. 选择您的配置
3. 启用"自动选择"功能

### Q: 小火箭会影响电池寿命吗？
A: 会有一定影响，建议：
- 不使用时关闭连接
- 定期重启应用
- 使用省电模式

### Q: 如何导出配置？
A:
1. 长按配置文件
2. 选择"分享"
3. 选择导出方式

## 安全建议

⚠️ **重要提示**：
- 仅从可信来源添加节点
- 定期更新应用
- 不要在公共 WiFi 上进行敏感操作
- 定期检查连接日志

## 性能优化

### 1. 选择合适的规则
- 根据需求选择规则
- 避免过于复杂的规则
- 定期更新规则

### 2. 调整 DNS
```
配置 → DNS → 选择 DNS 服务器
```

推荐 DNS：
- 8.8.8.8 (Google)
- 1.1.1.1 (Cloudflare)
- 119.29.29.29 (Tencent)

### 3. 启用 TUN 模式
```
配置 → TUN 模式 → 启用
```

优点：
- 更好的兼容性
- 更高的性能
- 支持更多应用

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 无法连接 | 检查节点有效性，尝试更换节点 |
| 速度慢 | 测试延迟，选择最快的节点 |
| 某些应用无法使用 | 调整规则，启用 TUN 模式 |
| 频繁断连 | 更换节点，检查网络稳定性 |

## 高级技巧

### 1. 使用脚本

- 自动选择最快节点
- 定时更新节点
- 自定义规则

### 2. 远程规则
```
配置 → 规则 → 添加远程规则
```

### 3. 定时任务
```
配置 → 定时任务 → 设置自动切换
```
