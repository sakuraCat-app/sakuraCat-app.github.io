## 安装和配置相关

### Q1: 如何安装 Clash？

**A:** 根据您的操作系统选择对应版本：

- **Windows**: [Clash for Windows](https://github.com/Fndroid/clash_for_windows_pkg)
- **macOS**: [ClashX](https://github.com/yichengchen/clashX)
- **Linux**: [Clash](https://github.com/Dreamacro/clash)
- **Android**: [Clash for Android](https://github.com/Kr328/ClashForAndroid)
- **iOS**: [Shadowrocket](https://shadowrocket.online/)

### Q2: 配置文件在哪里？

**A:** 不同平台的位置不同：

```
Windows: %APPDATA%\Clash\config.yaml
macOS: ~/.config/clash/config.yaml
Linux: ~/.config/clash/config.yaml
Android: /sdcard/Android/data/com.kr328.clash/files/
```

### Q3: 如何导入订阅链接？

**A:** 
1. 复制订阅链接
2. 打开 Clash
3. 进入配置管理
4. 选择"从 URL 导入"
5. 粘贴链接并确认

## 连接问题

### Q4: 无法连接到节点怎么办？

**A:** 按以下步骤排查：

1. **检查节点有效性**
   - 确认节点未过期
   - 尝试更换节点
   - 检查节点信息是否正确

2. **检查网络连接**
   - 确保网络可用
   - 检查防火墙设置
   - 尝试关闭代理重新连接

3. **重启应用**
   - 关闭 Clash
   - 等待 10 秒
   - 重新打开应用

4. **查看日志**
   - 打开 Clash 日志
   - 查找错误信息
   - 根据错误进行处理

### Q5: 连接后无法访问网站？

**A:** 可能的原因和解决方案：

| 原因 | 解决方案 |
|------|---------|
| DNS 配置错误 | 更换 DNS 服务器 |
| 规则配置错误 | 检查分流规则 |
| 节点限制 | 尝试其他节点 |
| 网站被屏蔽 | 使用 PROXY 规则 |

### Q6: 为什么某些网站无法访问？

**A:** 
1. 检查规则配置
2. 尝试添加网站到代理规则
3. 更换 DNS
4. 尝试不同的节点

## 性能问题

### Q7: 为什么速度很慢？

**A:** 
1. **选择更快的节点**
   - 进行延迟测试
   - 选择延迟最低的节点

2. **优化配置**
   - 启用 TCP 并发
   - 调整 DNS
   - 使用分流规则

3. **检查网络**
   - 检查本地网络
   - 关闭其他占用带宽的应用
   - 尝试有线连接

### Q8: 频繁断连怎么办？

**A:**
1. 更换节点
2. 检查网络稳定性
3. 更新 Clash 版本
4. 重置配置
5. 联系机场客服

### Q9: 如何减少流量消耗？

**A:**
1. **使用分流规则**
   - 国内网站直连
   - 减少不必要的代理

2. **禁用自动更新**
   - 关闭应用自动更新
   - 禁用视频自动播放

3. **使用低清晰度**
   - 观看视频时选择低清
   - 禁用图片加载

4. **定期清理缓存**
   - 清空浏览器缓存
   - 删除临时文件

## 安全和隐私

### Q10: 使用代理安全吗？

**A:** 
- ✅ 使用可信的机场和节点
- ✅ 定期更新应用
- ✅ 不在公共 WiFi 上进行敏感操作
- ✅ 检查机场的隐私政策

### Q11: 会被检测到吗？

**A:** 
- 使用现代协议（Trojan、VMess）隐蔽性较好
- 配置合理的规则可以降低风险
- 避免过度使用
- 不要进行违法活动

### Q12: 如何保护隐私？

**A:**
1. 使用可信的机场
2. 启用 DNS 加密
3. 定期更新节点
4. 不分享订阅链接
5. 使用强密码

## 机场相关

### Q16: 免费机场会被关闭吗？

**A:** 是的，免费机场经常被关闭。建议：
- 注册多个账户
- 定期检查可用性
- 准备备用机场
- 考虑升级到付费服务

### Q17: 如何选择合适的机场？

**A:**
1. 查看用户评价
2. 测试节点速度
3. 检查隐私政策
4. 对比价格和流量
5. 试用免费版本

### Q18: 付费机场值得吗？

**A:** 付费机场的优势：
- ✅ 更稳定的服务
- ✅ 更高的速度
- ✅ 更多的节点
- ✅ 更好的隐私保护
- ✅ 专业的客户支持
