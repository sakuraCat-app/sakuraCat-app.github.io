## 方式一：使用订阅链接（推荐）

### 什么是订阅链接？

订阅链接是一个特殊的 URL，包含多个节点信息。使用订阅链接可以：
- 一次导入多个节点
- 自动更新节点信息
- 简化配置过程

### 获取订阅链接

**步骤 1：注册机场账户**
```
访问机场网站 → 注册账户 → 登录
```

**步骤 2：获取订阅链接**
```
用户中心 → 我的订阅 → 复制订阅链接
```

### Clash 中导入订阅

#### Windows (Clash for Windows)

```
1. 打开 Clash for Windows
2. 点击"Profiles"选项卡
3. 点击"Create"按钮
4. 选择"Import from URL"
5. 粘贴订阅链接
6. 点击"Import"
7. 等待下载完成
```

#### macOS (ClashX)

```
1. 打开 ClashX
2. 菜单栏 → 配置 → 管理
3. 点击"+"按钮
4. 选择"从 URL 导入"
5. 粘贴订阅链接
6. 点击"确定"
```

#### Linux

```bash
# 编辑配置文件
nano ~/.config/clash/config.yaml

# 或使用命令行工具
clash -c <订阅链接>
```

#### Android (Clash for Android)

```
1. 打开 Clash for Android
2. 点击"配置"
3. 点击"+"按钮
4. 选择"从 URL 导入"
5. 粘贴订阅链接
6. 点击"导入"
```

#### iOS (Shadowrocket)

```
1. 打开 Shadowrocket
2. 点击"配置"选项卡
3. 点击"+"按钮
4. 选择"从 URL 导入"
5. 粘贴订阅链接
6. 点击"完成"
```

### 订阅链接问题排查

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 导入失败 | 链接格式错误 | 复制完整链接，不要包含空格 |
| 无法下载 | 网络问题 | 检查网络连接，尝试重新导入 |
| 节点为空 | 订阅过期 | 更新订阅链接或联系机场 |
| 更新失败 | 服务器问题 | 等待一段时间后重试 |

## 方式二：扫描二维码

### 获取二维码

1. 登录机场账户
2. 找到"分享"或"二维码"选项
3. 选择要分享的节点
4. 显示二维码

### 扫描导入

#### Clash 应用

```
1. 打开 Clash
2. 点击"+"或"添加"按钮
3. 选择"扫描二维码"
4. 对准二维码
5. 等待识别完成
```

#### 通用方法

```
1. 使用手机摄像头或二维码扫描应用
2. 扫描节点二维码
3. 复制识别结果
4. 在 Clash 中手动添加
```

### 常见问题

**Q: 二维码无法识别？**
A: 
- 确保光线充足
- 调整角度和距离
- 清洁摄像头镜头
- 尝试其他二维码

**Q: 扫描后无法导入？**
A:
- 检查应用是否支持该协议
- 尝试手动添加
- 更新应用版本

## 方式三：手动添加节点

### 节点信息获取

通常包含以下信息：
- 服务器地址（Server）
- 端口（Port）
- 加密方式（Cipher）
- 密码（Password）
- 协议类型（Protocol）

### 在 Clash 中手动添加

#### 编辑配置文件

```yaml
proxies:
  - name: "我的节点"
    type: ss
    server: 1.2.3.4
    port: 8388
    cipher: aes-256-gcm
    password: mypassword123
```

#### 使用 GUI 界面

```
1. 打开 Clash
2. 点击"+"或"添加"
3. 选择节点类型
4. 填写节点信息
5. 点击"保存"或"确定"
```

### 不同协议的配置示例

#### Shadowsocks

```yaml
- name: "SS节点"
  type: ss
  server: example.com
  port: 8388
  cipher: aes-256-gcm
  password: password123
```

#### Trojan

```yaml
- name: "Trojan节点"
  type: trojan
  server: example.com
  port: 443
  password: password123
  sni: example.com
  skip-cert-verify: false
```

#### VMess

```yaml
- name: "VMess节点"
  type: vmess
  server: example.com
  port: 443
  uuid: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
  alterId: 0
  cipher: auto
  tls: true
  servername: example.com
```

## 方式四：导入配置文件

### 获取配置文件

```
机场网站 → 下载配置 → 选择 Clash 格式
```

### 导入步骤

#### Windows

```
1. 下载配置文件
2. 打开 Clash for Windows
3. 点击"Profiles"
4. 点击"Open File"
5. 选择配置文件
6. 点击"Open"
```

#### macOS

```
1. 下载配置文件
2. 打开 ClashX
3. 菜单栏 → 配置 → 管理
4. 点击"打开文件"
5. 选择配置文件
```

#### Linux

```bash
# 复制配置文件到 Clash 目录
cp ~/Downloads/config.yaml ~/.config/clash/

# 重启 Clash
systemctl restart clash
```

## 导入常见问题

### Q: 导入后节点无法使用？

**A:** 
1. 检查节点是否过期
2. 验证节点信息是否正确
3. 尝试其他节点
4. 重启应用

### Q: 如何批量导入多个节点？

**A:**
- 使用订阅链接（推荐）
- 导入配置文件
- 手动逐个添加

### Q: 导入的节点在哪里查看？

**A:**
```
Clash → 代理 → 查看所有节点
```

### Q: 如何删除不需要的节点？

**A:**
```
1. 找到节点
2. 右键点击
3. 选择"删除"
4. 确认删除
```

## 最佳实践

### ✅ 推荐做法

1. **使用订阅链接**
   - 自动更新
   - 管理方便
   - 节点众多

2. **定期更新**
   - 每天更新一次
   - 及时获取新节点
   - 移除过期节点

3. **多机场备份**
   - 注册多个账户
   - 避免单点故障
   - 提高可用性

4. **定期测速**
   - 选择最快节点
   - 监控性能变化
   - 及时调整

### ❌ 避免做法

1. **分享个人订阅链接**
   - 可能导致账户被封
   - 流量被滥用
   - 隐私泄露

2. **使用过期节点**
   - 无法连接
   - 浪费时间
   - 影响体验

3. **忽视隐私设置**
   - 不检查隐私政策
   - 使用不可信机场
   - 泄露个人信息
